## Feature List 📃

- [x] File upload
- [x] File compression
- [x] File thumbnail
- [x] Temporary storage of files
- [x] File Recycle Bin
- [x] File Download
- [x] File editing
- [x] File sharing
- [x] File data management
- [x] User SSO

- [x] User Role Permissions
- [x] User disable enable
- [x] One click start of project
